Лекция №1
Концепция X SKILL: решение для быстрого обучения продавцов
Внутри мы рассмотрим общую концепцию продукта как комплексной школы продаж под ключ. Разберем основные проблемы бизнеса, которые решает система: высокая текучка новичков, отсутствие структурированного обучения, нагрузка на РОПов. Объясним, как система превращает человека без опыта в продавца за 3-4 недели. Сформулируем ключевое ценностное предложение, которое сейлз должен четко доносить до клиентов: "Готовая система обучения, с которой новички быстро понимают продукт и начинают продавать".
Лекция №2
Компоненты системы X SKILL и их ценность для клиента
Детально познакомимся с шестью основными компонентами системы. Разберем прямую линию обучения с визуализацией прогресса, теоретические курсы с автодоучиванием при ошибках, базу знаний с ИИ-поиском для мгновенного доступа к информации во время звонка. Также изучим систему разбора и оценки звонков, ИИ-тренажер для безопасной отработки навыков и аналитическую панель для контроля прогресса. Для каждого компонента объясним конкретную выгоду для клиента, используя формулу "Это означает, что вы получаете...", чтобы сейлз мог легко переводить функции в преимущества.
